Grficos Univariados da Log-Verossimilhana

Guilherme aqui fiz isso com o intuito de ver melhor como est o formato da verossimilhana e o quo deslocado ela est em cada parmetro, pois achei que nos grficos de contorno no muito claro.

Pra isso eu simulei um Processo Gaussiano com \(\sigma^2_1 = 1, \sigma^2_2 = 1, a = 2, \rho = 0.5, \nu_1 = \nu_2 = 0.5\) da mesma forma que simulado nos grficos de contorno.

Simulao do Processo

suppressPackageStartupMessages(library(BivMaternEstim))
source("univariate_loglike_funs.R")

n <- 100
coords <- matrix(runif(2*n), ncol = 2)
temp <- rnorm(2*n)
nus_vec <- c(0.5,0.5)
S <- sigma_assembler_biwm(sigmas = c(1, 1), a = 2, rho = 0.5,
                          nus = c(0.5, 0.5), coords_matrix = coords, combined = TRUE)

log_cd <- matrix(temp%*%chol(S) + rep(c(1,2), each = n), ncol = 2)

Grficos para \(\sigma^2_1\)

grid_sigma2_1_notzoomed <- construct_grid(
  seq(0.01,2.5,length.out = 100),

  0.2)

p_sigma2_1_notzoomed <- plot_univariate_llike("sigma2_1",grid_sigma2_1_notzoomed,log_cd,1)
